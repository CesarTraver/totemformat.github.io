# Totem Format Rules – v1.3

## 1. Overview
Totem Format adapts Pokémon TCG to a multiplayer environment (3 to X players), preserving the Prize Card system for full card compatibility.

## 2. Number of Players
3 to X players (3–6 recommended).

## 3. Game Setup
Each player starts with:
- A 100-card deck (singleton)
- One Totem Pokémon
- 6 face-down Prize cards
- A standard opening hand

(Mulligans as defined in the rules.)
